# Nerds
Сайт небольшой дизайн-студии.

## Основные характеристики разметки:

* Фиксированная сетка;
* Сетка выполнена на flexbox;
* Фреймворки не используются;
* Именование классов по БЭМ;
* Кроссбраузерность: IE 11+, Chrome, Firefox, Opera, Safari.

[Сайт дизайн-студии Nerds на GitHub Pages](https://igorbriukhovetskii.github.io/Nerds/)
